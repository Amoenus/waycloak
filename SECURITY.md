# Security policy

Waycloak is pre-alpha and does not yet have supported releases.

Please report suspected vulnerabilities privately through GitHub Security Advisories for this repository. Do not open a public issue containing credentials, provider configuration, public IP addresses tied to an individual, or an exploitable routing bypass.

The security model and explicit non-goals are documented in [`docs/security/threat-model.md`](docs/security/threat-model.md).
